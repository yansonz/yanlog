# Product Overview

## YANSO's Blog (yanlog)
개인 기술 블로그 - http://yanlog.yanbert.com

## 핵심 기능
- 다국어 지원 (한국어/영어)
- MDX 기반 콘텐츠 관리
- 정적 사이트 생성 (Static Export)
- 다크/라이트 테마
- 질문 태그: 독자가 궁금해할 내용을 질문 형태로 표시, 클릭 시 해당 단락으로 이동

## 콘텐츠 유형
- 블로그 포스트: 기술 글 및 개인 활동 아카이빙
- 페이지: About 등 정적 페이지

## 대상 사용자
- 블로그 소유자 (콘텐츠 작성)
- 방문자 (콘텐츠 소비)
